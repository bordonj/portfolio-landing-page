Project: Animal Shelter

Animal Shelter
Description

In another Epicodus lesson, we learned about using floats. Please see this page, which is the reference to this lesson. Some uses of floats include:

  • wrapping text around an image
  • Creating sidebars
  • Creating pages with multiple columns
  • Specifying the columns with either IDs or classes to customize each column as needed.

In this "animal shelter" example website, I designed a page with three columms for three different dogs. This was made to display the power of using columns and being able to change the background colors of the columns accordingly. I also used CSS to make the images and the names of the animals uniform to each other.

Tools used:
  • HTML
  • CSS
