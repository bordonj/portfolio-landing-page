Project: My Favorite Things

My Favorite things
Description
This webpage is about an imaginary person's favorite things. It was one of the first projects assigned by Epicodus to experience what it was like to customize a webpage with CSS. In this page, I pretended my name was Michael and listed a few imaginary favorite things, such as Michael's doggy relatives. Additionally, there is a Pet Page you can navigate to at the bottoms of the website to check out Michael's imaginary pet.
Tools used:
  • HTML
  • CSS
